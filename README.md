# Digital Grain Moisture Analyzer

This is a Next.js application for a digital grain moisture analyzer, built by Zaid Momin
## MVP Features

*   **Moisture Measurement**: Simulate measuring moisture content for various grains.
*   **Grain Selection**: Choose between Rice, Wheat, and Maize.
*   **Digital Display**: A clean interface to show moisture readings.
*   **AI-Powered Calibration**: Use a GenAI model to create calibration equations from sample data, improving measurement accuracy.

To get started, run `npm run dev` and open your browser to the specified port.
