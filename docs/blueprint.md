# **App Name**: GrainScan

## Core Features:

- Moisture Measurement: Accurately measure the moisture content of grains using a capacitive or resistive sensor.
- Grain Selection: Allow users to manually select the grain type (Rice, Wheat, Maize) using push buttons.
- Digital Display: Display real-time moisture percentage on an LCD screen.
- Calibration Tool: Utilize a simple regression model to calibrate moisture readings based on sample grain data; uses the calibration curves to more accurately calculate the mosture levels.

## Style Guidelines:

- Primary color: Soft blue (#89BCE8) to evoke trust and reliability.
- Background color: Light gray (#F0F4F7), nearly white, for a clean and professional look.
- Accent color: Pale green (#95E0B3), lighter than the primary, to provide soft contrast on the blue background.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modern and warm elements for headlines or body text.
- Use simple, clear icons to represent different grain types.
- Ensure a clear and intuitive layout for easy navigation and readability on the LCD screen.
